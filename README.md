# bts-ce-release
Boda Telecom Suite - CE release management scripts. This creates a zip file in the format: *bts-ce-vMAJOR.MINOR.PATCH.zip* e.g bts-ce-v2.0.0



## Usage 
```
./create_release.sh vMAJOR.MINOR.PATCH

```
